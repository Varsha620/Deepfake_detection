# DeepGuard: Deepfake Detection System 🛡️

A resume-worthy, sophisticated Deepfake Detection project utilizing state-of-the-art Deep Learning (EfficientNetB0) and a premium Streamlit web interface.

## 🚀 Features
- **SOTA Backbone**: Leverages EfficientNetB0 for high-accuracy feature extraction.
- **Real-time Detection**: Seamless Streamlit application for instant image analysis.
- **Premium UI**: Glassmorphism design with responsive elements and dark mode support.
- **Transfer Learning**: Optimized for the CIFAKE dataset (Real vs. AI-Generated).

## 🛠️ Technical Stack
- **Framework**: TensorFlow / Keras
- **Frontend**: Streamlit
- **Preprocessing**: OpenCV / NumPy
- **Backbone**: EfficientNetB0 (ImageNet weights)

## 📁 Project Structure
```bash
├── data/               # Place CIFAKE dataset here
├── models/             # Saved model checkpoints (.h5)
├── src/

│   └── model/
│       ├── model_utils.py # Architecture & Preprocessing
│       └── train.py    # Training Pipeline
├── requirements.txt    # Project Dependencies
└── README.md
```

## ⚙️ Getting Started

### 1. Prerequisites
Ensure you have Python 3.9+ installed.

### 2. Installation
```bash
pip install -r requirements.txt
```

### 3. Training (Optional)
To train the model on your own data (e.g., CIFAKE):
1. Download the CIFAKE dataset from Kaggle.
2. Place it in `data/cifake`.
3. Run:
```bash
python src/model/train.py
```

### 4. Running the App
```bash
streamlit run src/app/app.py
```

## 📊 Dataset
This project is designed to be trained on the **CIFAKE: Real and AI-Generated Images** dataset, which consists of 120,000 images (60,000 real, 60,000 generated by Stable Diffusion).
